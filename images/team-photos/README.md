This directory contains professional photos of the Awra Designs team members.

2. **sarah-bekele.jpg** - Photo of Sarah Bekele (Senior Interior Designer)

4. **bereket-fikre.jpg** - Photo of Bereket Fikre (Expert Graphic Designer)

## Photo Requirements:

- **Format**: JPG or PNG
- **Size**: 300x300 pixels minimum (square aspect ratio recommended)
- **Quality**: High resolution for crisp display
- **Style**: Professional headshots or business photos
- **Background**: Clean, professional backgrounds preferred
- **Lighting**: Good lighting to ensure clear, professional appearance

## How to Add Photos:

1. Save your team member photos in this directory
2. Name them exactly as listed above
3. The website will automatically display the photos
4. If a photo is missing, the system will fall back to emoji icons

## Fallback System:

If any photo fails to load, the system will automatically show the original emoji icon as a fallback, ensuring the team section always displays properly.

## Professional Tips:

- Use consistent lighting and background for all team photos
- Ensure all photos have the same aspect ratio (square)
- Consider using a professional photographer for best results
- Maintain consistent styling across all team member photos

## File Naming Convention:

- Use lowercase letters
- Separate words with hyphens
- Use .jpg or .png extension
- Keep filenames descriptive but concise
